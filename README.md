# HomeGuardians

## Project Overview
HomeGuardians is a React Native application designed to help users manage household tasks and reminders.

## Prerequisites
Before you begin, ensure you have the following installed:
- Node.js (v14 or higher)
- Xcode (v12+)

## Installation Instructions

### Clone the Repository
```bash
git clone https://github.com/YourUsername/HomeGuardians.git
cd HomeGuardians
Install Dependencies
=>bash

npm install
iOS Setup
Install CocoaPods:

=>bash

sudo gem install cocoapods
Install iOS dependencies:

=>bash

cd ios
pod install
cd ..
Run the app on iOS:

=>bash

npx react-native run-ios

Android Setup
Install Android Studio:

Download and install Android Studio.
Make sure to install the Android SDK and the necessary build tools during setup.
Set up Android Environment Variables: Ensure that the environment variables for Android SDK are set up correctly:

On macOS/Linux, add the following to your .bash_profile or .zshrc:
=>bash

export ANDROID_HOME=$HOME/Library/Android/sdk
export PATH=$ANDROID_HOME/emulator:$PATH
export PATH=$ANDROID_HOME/tools:$PATH
export PATH=$ANDROID_HOME/tools/bin:$PATH
export PATH=$ANDROID_HOME/platform-tools:$PATH
On Windows, set the environment variables through the system properties.
Run the app on Android:

Start an Android emulator or connect your physical device.
Run the app:
bash

npx react-native run-android
